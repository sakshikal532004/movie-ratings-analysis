# Movie Ratings Analysis Dashboard

This project analyzes movie ratings data using **PySpark** and **Databricks**, generating a dashboard of top-rated movies, genre analysis, and rating trends.

## Tools Used
- Databricks
- PySpark
- Python (Pandas, Matplotlib, Seaborn)

## Features
- Top 10 Movies by Average Rating
- Number of Movies per Genre
- Average Rating per Genre
- Rating Trends Over Years
- Top 5 Movies per Genre
- Professional Dashboard Visualization

## Outcome
- Actionable insights for recommendation systems and content strategy.
- Ready-to-share dashboard and analysis results.
